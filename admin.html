Area administrativa (versao estatica)

Esta versao hospedada no GitHub Pages funciona apenas com arquivos estaticos (HTML, CSS e JavaScript). Por isso, recursos que dependiam de PHP ou banco de dados foram adaptados para uso manual.

Como atualizar informacoes do site

  1. Edite o arquivo settings.json diretamente no GitHub ou no seu editor local. Ele controla nome da marca, telefone, enderecos e links sociais.
  2. Faça commit das alteracoes e publique na branch usada pelo GitHub Pages.

Como editar o cardapio

  1. Acesse o cardapio digital e clique em "Editar cardapio".
  2. Faça as mudanças desejadas (itens, precos, ingredientes). As alteracoes ficam salvas no navegador e voce pode exportar o HTML.
  3. Para publicar novidades, substitua o arquivo cardapio.html no repositório e adicione as novas imagens na pasta uploads/ ou assets/images/.

Precisa de funcionalidades com login ou upload direto? Considere manter uma versao PHP do projeto em um servidor com suporte a scripts (Hostinger, InfinityFree etc.).
