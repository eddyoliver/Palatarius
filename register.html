Cadastro indisponivel

O sistema de cadastro/usuarios fazia parte da versao em PHP do projeto. Na hospedagem estatica do GitHub Pages, os dados nao sao processados pelo servidor.

Use o cardapio digital e o arquivo settings.json para atualizar informacoes exibidas aos clientes. Se precisar de um painel com contas, publique o projeto em um host que suporte PHP e banco de dados.
